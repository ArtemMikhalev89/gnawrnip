# Gnawrnip demo

## Requirements

* Poltergeist
    * see [Installing PhantomJS](https://github.com/jonleighton/poltergeist#installing-phantomjs)

## Usage

    $ cd /path/to/gnawrnip/example
    $ bundle install --path vendor/bundle
    $ bundle exec rspec

memo: It raises an error for demo.

And open report file.

    $ open report.html
